Markdown to HTML Cheat-Sheet

Headers

  • Markdown: # Header 1
  • HTML: <h1>Header 1</h1>
  • Markdown: ## Header 2
  • HTML: <h2>Header 2</h2>

Emphasis

  • Bold
  • Markdown: **Bold Text**
  • HTML: <strong>Bold Text</strong>
  • Italic
  • Markdown: *Italic Text*
  • HTML: <em>Italic Text</em>

Lists

Unordered List

  • Markdown:
    • - Item 1
- Item 2
  - Sub-item 1
  • HTML:
    • <ul>
    <li>Item 1</li>
    <li>Item 2
        <ul>
            <li>Sub-item 1</li>
        </ul>
    </li>
</ul>

Ordered List

  • Markdown:
    • 1. Item 1
2. Item 2
   1. Sub-item 1
  • HTML:
    • <ol>
    <li>Item 1</li>
    <li>Item 2
        <ol>
            <li>Sub-item 1</li>
        </ol>
    </li>
</ol>

Links

  • Markdown: [Link Text](https://example.com)
  • HTML: <a href="https://example.com">Link Text</a>

Images

  • Markdown: ![Alt Text](https://example.com/image.jpg)
  • HTML: <img src="https://example.com/image.jpg" alt="Alt Text">

Blockquotes

  • Markdown:
    • > This is a blockquote
>> Nested blockquote
  • HTML:
    • <blockquote>
    This is a blockquote
    <blockquote>
        Nested blockquote
    </blockquote>
</blockquote>

Code

Inline Code

  • Markdown: `Inline Code`
  • HTML: <code>Inline Code</code>

Code Block

  • Markdown:
    • ```
code-block content
```
  • HTML:
    • <pre><code>
Code block content
</code></pre>

Horizontal Rule

  • Markdown: --- or *** or ___
  • HTML: <hr>

Tables

  • Markdown:
    • | Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
  • HTML:
    • <table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Cell 1</td>
            <td>Cell 2</td>
        </tr>
    </tbody>
</table>
